# TQC Python Study 🐍
# TQC Python 學習課程

Learn & Practice Python for TQC (Techficiency Quotient Certification) Exam preparation.

學習與練習 Python，為 TQC 認證考試做準備。

---

## 📚 Course Structure / 課程結構

This course is based on the **Python 程式設計概論** (Python Programming Introduction) curriculum and follows the TQC exam format for practice exercises.

### Lessons / 課程

| Lesson | Topic | Description |
|--------|-------|-------------|
| [01 - Python Basics](lessons/01_python_basics/) | 基礎 | print, input, variables, f-strings / 輸出、輸入、變數、格式化字串 |
| [02 - Selection Structures](lessons/02_selection_structures/) | 選擇結構 | if, elif, else, comparison & logical operators / 條件判斷、比較與邏輯運算子 |
| [03 - Loop Structures](lessons/03_loop_structures/) | 迴圈結構 | for, while, break, continue, nested loops / 迴圈、巢狀迴圈 |
| [04 - Functions](lessons/04_functions/) | 函式 | def, return, parameters, recursion, lambda / 函式定義、參數、遞迴 |
| [05 - Strings & Lists](lessons/05_strings_and_lists/) | 字串與串列 | String methods, list operations, comprehensions / 字串方法、串列操作、推導式 |
| [06 - Dictionaries & Applications](lessons/06_dictionaries_and_applications/) | 字典與綜合應用 | Dictionaries, sets, file I/O / 字典、集合、檔案讀寫 |

### TQC Practice Exam / TQC 模擬考試

| File | Description |
|------|-------------|
| [Mock Exam](tqc_practice_exam/mock_exam.py) | Auto-graded 10-question exam / 自動評分模擬考 |
| [Interactive Exam](tqc_practice_exam/interactive_exam.py) | Fill-in-the-code practice exam / 填寫程式碼互動考試 |

## 🚀 Getting Started / 開始使用

### Prerequisites / 前置需求
- Python 3.10+ installed ([python.org](https://www.python.org/))
- A code editor (VS Code, or any text editor with Python support)

### How to Run / 如何執行

```bash
# Clone the repository / 複製儲存庫
git clone https://github.com/devtamara07-tk/TQC-Pythong_Study.git
cd TQC-Pythong_Study

# Run any example / 執行任何範例
python lessons/01_python_basics/example_01_hello.py

# Run the mock exam / 執行模擬考試
python tqc_practice_exam/mock_exam.py

# Try the interactive exam / 嘗試互動考試
python tqc_practice_exam/interactive_exam.py
```

## 📖 Study Tips / 學習提示

1. **Follow the lessons in order** - Each lesson builds on the previous one
2. **Run every example** - Don't just read the code, run it!
3. **Try the exercises** - Practice is the best way to learn
4. **Modify the examples** - Experiment with changes to understand how things work
5. **Take the mock exam** - Test yourself before the real TQC exam
6. **Use the interactive exam** - Fill in the code yourself for hands-on practice
