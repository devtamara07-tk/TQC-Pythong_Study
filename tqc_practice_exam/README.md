# TQC Practice Exam 📋
# TQC 模擬考試

## About / 關於
These files simulate the TQC (Techficiency Quotient Certification) Python exam format.
這些檔案模擬 TQC Python 認證考試的格式。

## Files / 檔案

| File | Description |
|------|-------------|
| `mock_exam.py` | Auto-graded mock exam (10 questions) / 自動評分模擬考（10題） |
| `interactive_exam.py` | Interactive exam - YOU fill in the code! / 互動考試 - 你來寫程式！ |

## How to Use / 如何使用

### Mock Exam (watch and learn) / 模擬考試（觀看學習）
```bash
cd tqc_practice_exam
python mock_exam.py
```
This runs pre-written solutions and shows you TQC-style problems and answers.

### Interactive Exam (practice coding) / 互動考試（練習寫程式）
```bash
cd tqc_practice_exam
python interactive_exam.py
```
1. Open `interactive_exam.py` in your editor
2. Fill in the `TODO` functions
3. Run the file to check your answers
4. Try to get 100%!

## TQC Exam Categories / TQC 考試類別
The TQC Python exam covers these categories:
1. **Basic I/O** - print, input, variables / 基本輸入輸出
2. **Selection Structures** - if/elif/else / 選擇結構
3. **Loop Structures** - for, while / 迴圈結構
4. **Functions** - def, return, recursion / 函式
5. **Strings** - methods, slicing / 字串
6. **Lists & Tuples** - operations, comprehensions / 串列與元組
7. **Dictionaries & Sets** - key-value, set operations / 字典與集合
8. **File I/O** - read, write / 檔案讀寫
9. **Modules** - import, math, random / 模組
10. **Integrated Applications** - combining concepts / 綜合應用
