## Key Concepts / 重要概念

### Comments / 註解
```python
# This is a single-line comment / 這是單行註解
```

### print() function / print() 函數
```python
print("Hello, World!")
print("Hello", "World", sep=", ")  # Custom separator
```

### input() function / input() 函數
```python
name = input("What is your name? ")
```

### f-strings / 格式化字串
```python
name = "Alice"
print(f"Hello, {name}!")
```
